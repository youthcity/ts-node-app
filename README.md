# 使用Typescript和koa构建Node应用

## Usage

### Install 

```bash
yarn install

// or

npm install
```

### Production

```bash
npm run build
npm start
```

### Dev

方法一：

```bash
npm run dev
```

方法二：

```bash
npm run watch-server
```
